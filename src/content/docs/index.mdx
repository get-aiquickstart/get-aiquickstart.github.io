# Welcome to the AI QuickStart

In this mini workshop, you will learn the fundamentals of the Python programming language. But you won't waste time on boring projects list square the first integers. The demo prject for the Python QuickStart is a command line application to manage a cryptocurrency portfolio. It stores the transactions in a local database and even retrieves real time prices from the cloud.

If you already have a Python development environment set up, feel free to proceed to the [first module about Python](../curriculum/module02).

Otherwise, start with the [Onboarding](../curriculum/module01) module that will show you how to set up a development environment online and for free!
